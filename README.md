# 쇼핑몰 웹 만들어보기

    H&M 웹 사이트를 참고해 해당 사이트의 제품 정보들을 이용해 api호출, 웹 디자인을 직접 함.

# 기술 스택

    React, HTML, CSS, JavaScript, JsonServer, react-query

# 개선사항

    1. 로그인 페이지 (상태 관리로 해결)
        - useState()를 이용해서 authenticate, true || false 로 구분해 구현
    2. redux로 전역관리 ()
    3. 반응형 웹 디자인
    4. react-query를 이용해서 api정보 호출
